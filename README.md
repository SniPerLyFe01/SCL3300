# SCL-3300 C-driver using stm32 controller/(HAL libraries)

## Setup
* NUCLEO board STM32H743ZIT6
* SCL-3300 Chip carrier board

## Configuration
Change the SPI handle and Chip_Select pin in the header file of the sensor driver.
